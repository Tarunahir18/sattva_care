# 🌿 Sattva Care — E-Commerce Website

> **Natural. Safe. Effective.**

A full-stack e-commerce website for **Sattva Care** selling Herbal Shampoo, Hair Oil, and Soap.

## 🛠 Tech Stack

| Technology | Usage |
|---|---|
| **React** | Frontend SPA with hooks, context, routing |
| **Bootstrap 5** | Grid, navbar, modals, forms, badges, cards |
| **CSS** | Custom animations, glassmorphism, themes, gradients |
| **JavaScript** | ES6+, async/await, fetch API, array methods |
| **Node.js + Express** | Backend REST API, CRUD operations |

## 🚀 How to Run

### 1. Start the Backend
```bash
cd backend
npm install
node server.js
```
Backend runs on: **http://localhost:5000**

### 2. Start the Frontend
```bash
cd frontend
npm install
npx vite --port 3000
```
Frontend runs on: **http://localhost:3000**

## 📦 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/products` | Get all products |
| GET | `/api/products/:slug` | Get single product |
| POST | `/api/orders` | Place an order |
| GET | `/api/orders/:id` | Get order details |
| GET | `/api/reviews/:productId` | Get reviews |
| POST | `/api/reviews` | Submit a review |
| POST | `/api/contact` | Submit contact form |
| POST | `/api/coupon/validate` | Validate coupon |
